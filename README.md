WARP
====


Simple http transparent proxy made in python 2.7.3 (can be run with pypy)

This is proof-of-concept code.


## Dependency
* python 2.7.3
* gevent >= 0.13.8

## How to use
1. run with python interpreter

        $ python warp.py

2. set browser's proxy setting to 

 [http proxy] host: 127.0.0.1 / port: 8800

3. ???

4. PROFIT!

### Command help
$ python warp.py --help
